# studionet
Social networking for documents


### Defining environment variables
```
export DB_USER="your_db_user"
export DB_PASS="your_db_pass"


export PORT="port"
export SERVER_URL ="server_url"
export SITE_URL ="site_url"

```

### Run app
```
npm start
```
or
```
nodemon
```
